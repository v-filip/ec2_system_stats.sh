### ec2_system_stats.sh



Simple script that I personally use on an ec2 instance hosting my webserver



#### Usage 



```
./ec2_system_stats.sh --h [OR]

~/.scripts ❯ ./ec2_system_stats.sh --help [OR]

~/.scripts ❯ ./ec2_system_stats.sh help

-v --verbose

    Prints more info!
```
